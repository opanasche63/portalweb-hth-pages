# portalweb-hth-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容互不关联，各自独立运行，不针对任何特定域名或网站。

## 项目简介

这是一个静态页面集合仓库，主要用途是存放各类 HTML 文件。每个页面均为独立单元，不依赖本仓库以外的外部资源（除非页面本身有引入）。仓库本身不提供任何形式的服务或跳转，仅作为页面文件的存储与版本管理。

## 目录说明

```
portalweb-hth-pages/
├── README.md          # 本文件
└── pages/             # 存放所有 HTML 页面（按需组织子目录）
```

- `pages/` 是主要目录，所有归档的 HTML 页面均存放于此。可根据需要建立子目录进行分类。
- 每个页面应保持独立，避免与其他页面产生交叉引用。

## 页面归档说明

- 所有页面均为静态 HTML 文件，后缀名为 `.html`。
- 页面可以包含 CSS、JavaScript 或图片等资源，但建议将资源内联或与 HTML 文件一同存放。
- 归档时请确保页面内容完整，无外部依赖断链。
- 文件名建议使用英文小写字母、数字和连字符，避免特殊字符。

## 维护说明

- 本仓库仅供归档发布，不提供问题反馈或功能更新保证。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录下，并确保文件命名不与已有文件冲突。
- 不建议修改已归档页面的内容，除非是修复明显的显示错误或链接失效。
- 本仓库不设自动化部署或持续集成，更新后需手动推送至远程仓库。

## 其他

- 本仓库不收集、不存储任何用户信息。
- 页面内容不代表仓库维护者的立场或观点。
- 如有疑问，请参考 GitHub 文档或仓库历史记录。
